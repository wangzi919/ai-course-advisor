# 🎓 NCHU AI Course Advisor

國立中興大學（NCHU）智慧型校園顧問系統，以本地 LLM（Ollama）驅動，整合課程、圖書館、校務規章等多元資訊查詢工具。

---

## 📡 可取得的資訊

### 📚 課程資訊

- 課程名稱、開課系所、課程類別（必修／選修）
- 授課教師、課程網址
- 課程目標、課程簡述
- 每週授課內容（第 1～18 週）
- 評分方式
- 全校課程統計（各系所課程數、必選修比例、教師開課數排行）

---

### 📖 圖書館 — 館藏與借閱

- 書名、作者、館藏連結（Primo API）
- 個人借閱紀錄：借出日期、歸還日期、書名、作者、條碼

---

### 🎟️ 圖書館 — 活動

- 活動標題、日期、時間、地點
- 主講人、名額、報名連結
- 即將舉辦的活動清單

---

### 💻 圖書館 — 數位學習

- 數位課程名稱、類別、來源平台
- 熱門課程排行（依瀏覽數）
- 課程分類與各分類數量

---

### 📋 圖書館 — 使用指南與規則

- 圖書館各服務章節內容（借閱、還書、館際合作等）
- 指南中的規則表格
- 借閱規則自然語言問答（Gemini Pro 2.5 驅動）

---

### 🕐 圖書館 — 開放時間

- 各樓層／空間開放時段
- 24 小時開放空間清單
- 開放時間資料更新狀態

---

### 📣 校園活動公告

- 活動／公告標題與內文摘要
- 活動日期、發布日期、來源連結
- 即將到來的校園活動（可指定查詢天數）

---

### 📦 課程模組

- 所有可選課程模組名稱
- 各模組詳細說明
- 模組修課規定全文

---

### ⚖️ 校規查詢

- 依自然語言查詢相關校規條文（語意搜尋）
- 所有校規文件清單與來源

---

## 🚀 快速啟動

```bash
# 執行所有工具探索
python ste_runner.py

# 只針對特定工具
python ste_runner.py --filter course_search_by_keyword

# 自訂模型與參數
python ste_runner.py --model qwen3:8b --episodes 15 --turns 5
```

## 🧠 需求

- Python 3.10+
- [Ollama](https://ollama.com/)（例如 `ollama pull qwen3:8b`）
