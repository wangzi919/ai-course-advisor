# 🎓 NCHU AI Course Advisor

國立中興大學（NCHU）智慧型校園顧問系統，以本地 LLM（Ollama）驅動，整合課程、圖書館、校務規章等多元資訊查詢工具。

---

## 📡 可取得的資訊

### 📚 課程資訊

- 課程名稱、開課系所、課程類別（必修／選修）
- 授課教師、課程網址
- 課程目標、課程簡述
- 每週授課內容（第 1～18 週）
- 評分方式
- 全校課程統計（各系所課程數、必選修比例、教師開課數排行）

---

### 📖 圖書館 — 館藏搜尋

- 書名、作者、館藏連結（Primo API）

---

### 🎟️ 圖書館 — 活動

- 活動標題、日期、時間、地點
- 主講人、名額、報名連結
- 即將舉辦的活動清單

---

### 📋 圖書館 — 借閱規則

- 借閱規則自然語言問答（Ollama 本地模型驅動，無需金鑰）

---

### 🕐 圖書館 — 開放時間

- 各樓層／空間開放時段
- 24 小時開放空間清單
- 開放時間資料更新狀態

---

### 📣 校園活動公告

- 活動／公告標題與內文摘要
- 活動日期、發布日期、來源連結
- 即將到來的校園活動（可指定查詢天數）

---

### 📅 學校行事曆

- 行事曆事件搜尋（支援關鍵字、日期範圍、類別、月份篩選）
- 即將到來的行事曆事件查詢（可指定未來天數）
- 依事件類別查詢（包含：開學、放假、考試、選課、註冊、畢業、活動、行政等常見分類）
- 快速取得特定期間事件（今天、特定月份）或特定主題事件（放假日、考試日、選課與註冊日）

---

### 📦 課程模組

- 所有可選課程模組名稱
- 各模組詳細說明
- 模組修課規定全文

---

### ⚖️ 校規查詢

- 依自然語言查詢相關校規條文（語意向量搜尋，使用 bge-m3）
- 所有校規文件清單與來源

---

### 👩‍🏫 教師與授課歷史

- 綜合教師名單搜尋（依姓名、系所、學院、研究專長）
- 課程師資反查（查詢某位老師過去與現在的實際授課清單）

---

## 🚀 快速啟動

```bash
# 執行所有工具探索
python ste_runner.py

# 只針對特定工具
python ste_runner.py --filter course_search_by_keyword

# 自訂模型與參數
python ste_runner.py --model qwen3:32b --episodes 5 --turns 5
```

## 🧠 需求

- Python 3.10+
- [Ollama](https://ollama.com/)

```bash
# 主要對話模型
ollama pull qwen3:32b

# 中文語意搜尋 Embedding 模型（校規查詢）
ollama pull bge-m3
```

## ⚙️ 環境變數

所有工具均在本地端運行，**不需要任何外部 API 金鑰**。

| 變數 | 預設值 | 說明 |
|------|--------|------|
| `EMBEDDING_API_URL` | `http://localhost:11434/api/embed` | Ollama bge-m3 的 embedding 端點，用於校規查詢的語意搜尋。若 Ollama 部署在非預設位址，才需要手動設定。 |

> **注意**：若未執行 `ollama pull bge-m3`，校規查詢（`rule_search_by_query`）工具將無法使用向量檢索功能。
